**数据结构**：数据在计算机中的存储方式。将数据具体实际的逻辑结构在物理存储中表现出来



线性结构：一对一

所有数据排成一条直线，前后只有一个邻居

数组、链表、栈、队列

特点：有唯一第一个、最后一个元素

（2）树形结构：一对多

一个节点分出多个子节点，像树分叉

二叉树、二叉搜索树、堆、红黑树

用途：文件目录、层级菜单、排序查找

（3）图形结构：多对多



线性结构：

一.线性表

实际数据是线性逻辑结构，有前去和后继。增删查改。

      1. 顺序表

      	采用数组的形式存储数据，需要定义一个变量标记顺序表长度。（顺序表只是操作后数组中有存储实际元素的部分，整个数组不一定是一个顺序表）

      	缺：删除和添加元素需要挨个遍历移动，效率低。顺序表长度是确定的，扩大长度不方便。

      	优：可以通过下标直接访问。



      2. 链表（头节点可带可不带）

逻辑上连续，物理上不是连续的 。每个节点包含数据域和指针域，指针域指向其它节点地址。第一个存储实际元素的节点称为首元节点。为了操作方便（让定义的函数，对不同的节点操作，均能相同使用），通常会定义一个头节点，头节点指针域为首元节点或NULL，数据域一般不存放数据。	链表中第一个节点存储的位置叫做头指针，（包括头节点）。一般用头指针的名字表示整个链表。

      	缺：不能通过下标直接访问，需要遍历整个链表。

      	优：删除和添加元素可以直接插入不需要挨个移动.

      	（1）单链表

      		链表的每个节点只包含一个指针域。为节点指向空。

      	（2）双向链表

      		每个节点包含两个指针域分别指向下一个和上一个节点，空链表的连个指针均指向自己。

      	（3）循环链表

      		尾节点指向头节点。





二.栈（物理内存上的栈是用来存储临时数据的）

后进先出 的线性数据结构，只能在同一端存，取数据。有栈顶和栈底。不能·对栈中间的元素进行操作，只能操作栈顶出栈和入栈。（一种特殊的线性表）能方便实现某些特殊操作。

      1.顺序栈

      采用数组存储，需要定义一个变量栈顶指针表示栈顶（顺序栈只包括栈顶到栈底的部分实际元素，整个数组不一定是一个顺序栈）

      缺：空间有限

      2.链式栈

      与链表类似，采用节点进行链式存储。不过不同的是只能对栈顶进行操作，无法操作栈中间的元素。



三.队列

先进先出的线性数据结构。一端存数据，另一端取数据。有队首（出队）和队尾（入队）。不能·对中间的元素进行操作，只能操作存，取数据。（一种特殊的线性表）能方便实现某些特殊操作（某些计算机的排列需求）



      1.普通队列

      	（1）顺序队列

      	采用数组存储，定义队首和队尾，分别用首指针和队尾指针表示。数组本身并不是队列，队列是其在数组中占用的部分。出队和入队可以通过移动首指针和尾指针实现（不用遍历，效率更高）。能方便实现某些特殊操作。

      	解决假溢出空间浪费：采用循环数组实现循环队列，下标加一时进行模运算（r=(r+1)%MAX）。

      	判空和判满：l==r时队列为空	此外借助三种方式判空（1）引入sum，记录队列中数据的数量，f==r\&\&sum==0  空，f==r\&\&sum==MAX 满。（2）引入变量flag，记录刚刚的操作是什么 flag==0刚刚执行过出队操作 ，falg==1刚刚执行过入队操作，flag的初值为0。f==r\&\&flag==0 空，f==r\&\&flag==1 满。（3）牺牲一个空间判满，认为MAx这么大的数组，装MAX-1个数据就满了。f==r 空，f==（r+1）%MAX 满。	（1）和（2）要进行额外操作，推荐使用（3）

      	（2）链式队列

      	基于单链表实现，头指针做队首指针，在整一个队尾指针指向队尾。初始化时让队首和队尾指针指向头节点，尾插入队移动尾指针，删除首元节点出队尾指针不动，注意当删除最后一个节点的时候应该让队尾指针指向头节点。f==r 或 f->next==NULL 为空

      

      2.双端队列

      在两端都可以进行插入和删除的一种特殊线性表，有前端和后端，入队时前端数据放在后端数据的前面。入队时允许随时出队和只能全部入队后一起出队的顺序不一样

      	（1）顺序实现

      	采用循环数组存储，同上解决假溢出问题。定义一个左端指针和右端指针，初始化时左端指针和右端指针，只有某一边的指针才能指向真正数据，即存储时先移动指针后放数据，不能两端指针指向同时为真正数据（数据会被覆盖），也不能两端指针指向同时不为真正数据（逻辑上的线性存储在物理上不再相邻）。

      	左端出队：l=（l+1）%MAX  	右端出队：r=（r-1+MAX）%MAX

    	判空和判满：与上述循环队列判断类似，有三种方案，只有当l==r时为空

      	（2）链式实现
        采用双向链表实现，此时链表的头节点的数据域存有数据并且在存储时头节点不一定在队首。定义左端指针和右端指针，初始化时左端指针和右端指针均指向头节点，注意只有一端的指针指向为真正数据，另一端指针指向为真正数据的下一个。当左端指针==右端指针时为空
